# UNP_PROJECT_BI_WITH_POWER_BI_E-COMMERCE_BRAZILIAN_ANALYSIS


 I did this project in collaboration with UNP, where I created an e-commerce dashboard using Power BI to understand overall business performance. The dashboard shows important details about payments, revenue, customers, sellers, orders, and reviews in a clear and organized way. It highlights key values like total revenue, total orders, total customers, and average values. With the help of filters and visuals, it is easy to analyze data by time, location, and product category, and identify trends such as sales changes, customer distribution, and product performance. The project also includes some business problems and simple suggestions to improve performance, like reducing order cancellations, improving low-performing products, and increasing customers in less active areas. Overall, this dashboard helps in understanding data easily and supports better decision-making.
